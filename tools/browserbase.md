Browserbase provides managed browser infrastructure for agents that need to act on the web at scale. The current platform supports standard automation frameworks, persistent contexts, live debugging, and stealth features like proxies, CAPTCHA solving, and fingerprint controls.

## Features

- Compatible with Playwright, Puppeteer, Selenium, and Chrome DevTools Protocol
- Fast serverless browser provisioning and large-scale session fleets
- Contexts API for persistent cookies and browser state across sessions
- Live View, session recording, logging, and source capture for debugging
- Managed stealth stack with CAPTCHA solving, residential proxies, and browser fingerprint controls

## So agents can...

- Run large fleets of browser tasks without self-hosting Chromium infrastructure
- Debug failed agent sessions with replays, logs, and live browser visibility
- Persist authenticated web sessions for research, operations, or transaction workflows
