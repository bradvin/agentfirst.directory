Firecrawl is a web data API designed for LLM and agent workflows. Its current API surface covers scrape, crawl, map, search, change tracking, browser actions, and agent-oriented collection of clean markdown, JSON, HTML, screenshots, and summaries.

## Features

- Scrape endpoints that return markdown, JSON, HTML, screenshots, links, images, and summaries
- Crawl and map endpoints for enumerating and traversing entire sites
- Search endpoint for web discovery with optional scrape-on-search workflows
- Browser actions and advanced scraping controls for dynamic sites and PDFs
- Agent and change-tracking capabilities for ongoing monitoring workflows

## So agents can...

- Extract structured product, company, docs, or news data from sites that are messy to parse manually
- Build knowledge bases by mapping and crawling large documentation or competitor sites
- Watch target pages for changes and feed updates into alerting or memory systems
