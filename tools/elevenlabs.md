ElevenLabs combines expressive speech models with a conversational agent platform. The current product supports voice and chat agents, external tool calls, telephony, multilingual conversations, custom voices, and developer APIs for embedding real-time audio experiences.

## Features

- Text-to-speech models optimized for both expressive output and low-latency conversational use
- Speech-to-text models with realtime transcription support
- Multimodal agents that can talk, type, and call tools during a conversation
- Telephony and WebRTC deployment options for phone, web, and app experiences
- Custom voices, voice cloning, knowledge-base grounding, testing, and analytics

## So agents can...

- Hold natural voice conversations for support, sales, scheduling, or training workflows
- Speak in branded or cloned voices while still grounding answers in company data
- Combine speech with tool use so a phone or web voice agent can actually complete tasks
