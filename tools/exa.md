Exa is a search and research API built for agent workflows. Its current platform includes fast search, deep multi-step research, structured outputs, field-level grounding, and specialized web indexes for companies, people, code docs, finance, and news.

## Features

- Fast search modes tuned for low-latency web grounding in agent loops
- Deep research endpoints that plan, search, read, and synthesize multi-source answers
- Structured output schemas for typed JSON instead of free-form text
- Content retrieval, summaries, and highlight support in API and SDK workflows
- Specialized indexes for companies, people, code docs, financial data, and news

## So agents can...

- Run semantic web search that is better aligned with downstream LLM reasoning than keyword SERPs
- Produce structured enrichments and reports with citations for each field
- Support research-heavy workflows where dozens of searches and reads need to be orchestrated reliably
