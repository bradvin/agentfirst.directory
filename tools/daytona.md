Daytona is a secure execution platform for AI-generated code and long-running agent workflows. Its current docs highlight OCI-based sandboxes, per-sandbox networking controls, shared volumes, and full computer-use environments across Linux, Windows, and macOS.

## Features

- Stateful sandboxes created from OCI-compliant images with programmatic APIs and SDKs
- Full compute environments where agents can install packages, run servers, and manage processes
- Per-sandbox network stacks and firewall controls for tighter isolation
- Shared volumes that persist across sandbox lifecycles and can mount into multiple runtimes
- Computer-use sandboxes for Linux, Windows, and macOS automation

## So agents can...

- Run, debug, and test generated code in isolated environments with filesystem and terminal access
- Reuse shared datasets or artifacts across many sandboxes without re-downloading everything
- Execute long-running automation or desktop tasks while keeping the host system out of scope
